# Knowledge Compression

Knowledge Compression is a Next.js / TypeScript application for turning nonfiction books into substantial long-form compressed curricula.

Before adding or expanding book modules, read the repository content standard:

- [Content Standard](docs/content-standard.md)
- [Codex Handoff](docs/codex-handoff.md)

Future curricula should be prose-first, book-specific, and deep enough to feel like a premium study guide or compressed masterclass, not a quick summary or microlearning outline.

## Development

```bash
npm install
npm run dev
npm run lint
npm run build
```
